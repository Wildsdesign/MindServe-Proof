# AGENTS.md - MindServe Proof

This repository is a curated public-proof artifact for MindServe. It is not the
production repository and must not become a runnable clone.

## Rules

- Do not copy source code from the private MindServe repo.
- Do not add inventory, recipe, production-board, pull-sheet, costing, or import
  internals.
- Do not add real URLs, resource names, workflow files, schema details, secrets,
  or environment files.
- Keep this repo docs-first and suite-level. Small sanitized suite-level code
  excerpts are allowed only when they avoid production internals.

## Purpose

Show engineering judgment:

- suite architecture
- operator-specific surfaces
- facility-scoped thinking
- Azure-first delivery posture
- ADR discipline
- agentic operating model
- intelligence-layer direction

Private MindServe remains the implementation source of truth.
