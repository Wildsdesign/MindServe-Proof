# Public Boundary

MindServe has real market value. This proof repo is intentionally strategic and
architectural rather than cloneable.

## Safe To Show

- suite map
- operator-surface descriptions
- high-level Azure posture
- facility isolation principle
- ADR discipline
- AI-assisted operating model
- intelligence-layer strategic direction
- small suite-level code excerpts that do not expose implementation internals
- non-sensitive screenshots later, if explicitly reviewed

## Not Safe To Show Here

- production source code
- detailed inventory engine behavior
- recipe engine behavior
- production board or pull-sheet internals
- costing and import internals
- full endpoint maps
- database schema
- deployment files
- customer/facility data
- secret names or resource names

## Companion Proof Repos

The deeper product-specific proof belongs in separate artifacts:

- Grocery proof: deeper production/recipe/inventory architecture, curated and
  protected.
- MealMind proof: patient-facing ordering, voice, PHI-minimizing architecture.

MindServe proof stays at the suite level.
